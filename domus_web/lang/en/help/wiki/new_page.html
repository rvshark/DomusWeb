new page block help

New page isn't a necessary block. It's useful for wikis that doesn't need an structurated index ( wwwroot.'/help.php?module=wiki&file=index.html">index block'; ?> ) because is better to use a glossary ( wwwroot.'/help.php?module=wiki&file=list_pages.html">list pages block'; ?> .

The effect of create a new page is the same as click on an internal link in the page content. If the page already exists dfwiki will display the view tab for this page, but if there aren't any page with this name, dfwiki will display the edit tab.

New page can be considered as a direct acces link. You write a pagename in the form and dfwiki redirects to this page as if you click on that page link.
