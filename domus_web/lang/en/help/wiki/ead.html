Administration tools block help

This block called EAD contains two type of useful tools for your dfwiki separated by a line:

  • Estadistical Tools: contains useful information of your dfwiki. By clicking in the tool name, the wanted information will be dysplayed in the main space.
  • Administration tools: this tools are only available for teachers and administrator and contains the functions to delete, rename or make editable the current page.

Estadistical Tools

In the current version of dfwiki this point contains six tools:

  • Most viewed pages: displays a list with all pages ordered by the times they're visited.
  • More updated pages: displays all pages oredered by it's last modified time.
  • Newest pages: displays all pages list ordered by it's creation date.
  • Wanted pages: displays the list of wanted pages like wwwroot.'/help.php?module=wiki&file=wanted.html">wanted pages block'; ?> .
  • Orphaned pages: displays the list of orphaned pages like wwwroot.'/help.php?module=wiki&file=orphaned.html">orphaned pages block'; ?> .
  • Activest Users: Return the list of all users that edit some page ordered by the number of his contributions.

Administration tools

This point contains some tools only available for teachers ans administrators:

  • Delete page: this option will ask you for delete a whole page content, including it's old version. Deleting a page will be irreversible.
  • Update page: this tool si for renaming pages. Renaming a page name will automatically redirect it's synonymous and, if it's the first page of the dfwiki, will rename the first page too. Althrough, renaming a page will NOT redirect all internal links, this means that if you rename a page, all old links will be erroneous. To detect if there's some link a renamed page, you only have to find the old name into the wwwroot.'/help.php?module=wiki&file=wanted.html">wanted pages'; ?> .
  • Disable/Enable Edition: this tool change the actual 'editable' propierty of the current page. Modifing the editable propierty of a page, you make it editable/undeditable for the students. Use this tool when you want to put an static text (for the students) into the dfwiki.
