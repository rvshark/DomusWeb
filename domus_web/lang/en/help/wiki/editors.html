Editor Selection

dfwiki can work with three diferents edit systems:

  • dfwiki: Original dfwiki (inspired in wikipedia style)
  • ewiki: Follows dfwiki style but adapted with the old Moodle wiki syntax
  • html editor: This system uses the standard Moodle edit area (ms office look like)

dfwiki system

Choosing this system, you wiki will use the same syntax as wikipedia. wikipedia is the current hugest wiki in the world and contains millions of documents inside.

If your students are used to work with other wikis on the net, may be they will preffer this system because it follow the most standard wiki format.

ewiki system

Choosing this system, the syntax for your documents will be like the old Moodle wiki module. This system is normally preferred for student that works with Moodle in the past, but the ewiki syntax can be a bit ambiguous

html editor

If you want to use the standard Moodle edition (a wysiwyg editor similar to ms word or open office text editors), this is your system. Use this system will cancell all wiki markup except internal and external links because the editor offers you visual solutions for formatting your documents.
