Attach and Upload help

In dfwiki you can attach files to the content of a page and upload files to the server.

Attach Files:

If the users have permissions to attach files the 'Insert File' tree will be shown. You can insert into the content any file uploaded to the server. To view the list of available files you only have to click on the plus icon.

Once you find the file in the list you only have to click on the name for insert it into the text (in the text area will appear something like '[[attach:filename]]').

If you aren't sure of which file want, you can click on the preview icon for open a pop up window or download it.

If you have a teacher or administrator, you can deleted uploaded files just by click on the delete icon.

Images files (gif, jpg, bmp and png) and flash movies (swf) will be inserted visually in the document. All other files will be displayed as a link to download.

Upload Files:

If you have permission to upload files to the server, you will see a standard upload form. Just browse the file and click on upload. If the file already exists on the server, a red message will appear in the top of the editor. Otherwise, the new file will appear automatically in the attach file list.
