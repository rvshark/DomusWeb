Introdução

Moodle é um pacote de software para produzir cursos via internet e sites web. Está em contínuo desenvolvimento sendo projetado para suportar uma estrutura socioconstrucionista da educação.

Moodle é fornecido livremente como software Livre (sob a GNU Public License). Basicamente isto significa que o Moodle é Copyright, mas que você tem liberdades adicionais. É permitido copiar, utilizar e modificar concordando-se a: fornecer os programas fontes para os outros, não modificar ou remover a licença original e os copyrights, e aplicar esta mesma licença para qualquer trabalho derivado. Leia a licença para detalhes completos e por favor contate o detentor do copyright diretamente se tiver quaisquer perguntas.

Moodle rodará em qualquer computador que tenha PHP, e pode suportar muitos tipos de banco de dados (particularmente o MySQL).

A palavra Moodle foi originalmente um acrônimo de Modular Object-Oriented Dynamic Learning Environment, que é na maioria das vezes útil para os programadores e teóricos em educação. É também um verbo que descreve o processo de percorrer meandros com paciência através de algo, fazendo coisas quando ocorre fazê-las, um agradável reparo que frequentemente direciona à perspicácia e à criatividade. Como tal, isto se aplica a ambas as maneiras que o Moodle foi devenvolvido, e à maneira que um estudante ou professor pode enfocar o estudo ou o ensino em um curso on-line. Qualquer um que utilizar o Moodle é um Moodler.
