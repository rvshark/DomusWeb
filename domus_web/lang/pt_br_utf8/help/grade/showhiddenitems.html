Mostrar itens ocultos

Especifica como as notas ocultas são mostrados. Selecionando Esconder, a nota é totalmente oculta. Selecionando Mostrar, a linha da nota oculta é visualizada com outra cor e a nota é oculta. Escolhendo "Somente ocultos até", items que tem uma data definda para ocultamento são exibidas com outra cor e as notas ocultas até a data escolhida.
