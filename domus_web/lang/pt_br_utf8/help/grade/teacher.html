Avaliações

A primeira vez que o Administrador de Avaliações é usado a configuração padrão é a modalidade simples. Você vai ver todos os estudantes, as notas de cada atividade avaliada e o total de cada estudante.

  • A lista pode ser ordenada pelo nome ou sobrenome do estudante clicando os links correspondentes na coluna dos estudantes. A ordem padrão é a lista por sobrenome.
  • Os nomes dos estudantes são links que mostram apenas aquele estudante e as suas notas. Isto é útil quando você está consultando os resultados das avaliações junto com um estudante, para proteger os dados pessoais dos demais estudantes.
  • O botão no alto da página cria um documento em formato Excel ou Texto separado por tabulações para que você possa baixar os dados no seu computador.
  • No modo normal o título principal é 'Avaliações'. Em modo de configuração avançada o título mostra o nome da categoria que você está acessando.
  • No meio são mostrados todos os itens avaliados de um curso (ou de uma categoria específica em modo avançado). Os nomes das atividades são links que abrem as páginas de configuração de cada item específico.
  • A coluna do Total é posicionada à direita. As duas flechas ao lado do título do total mudam a ordem de visualização para crescente ou decrescente.
  • O link 'stats' ao lado do título do total (ou da categoria) abre uma nova janela com estatísticas sobre os resultados totais da classe.

Como modificar a configuração do Administrador de Avaliação. O link que ativa a janela de configuração normalmente é visualizado entre aspas. Em cada página específica são disponíveis informações e ajuda.

  • Normal:

    • Ativar o modo avançado: "Configurar Preferências"
    • Mudar a frequência de repetição dos títulos das colunas do relatório: "Configurar Preferências"

  • Avançado:

    • Definir as colunas que serão mostradas: "Configurar Preferências"
    • Definir a modalidade de cálculo da nota final: "Configurar Preferências"
    • Designar a categoria de um item avaliado: "Configurar Categorias"
    • Definir a curva de aproximação da nota de uma tarefa: "Configurar Categorias"
    • Designar um item avaliado como crédito adicional: "Configurar Categorias"
    • Adicionar uma categoria: "Configurar Categorias"
    • Cancelar uma categoria: "Configurar Categorias"
    • Definir o peso das avaliações: "Configurar Pesos"
    • Descartar as X piores avaliações de uma categoria: "Configurar Pesos"
    • Adicionar bônus a uma categoria: "Configurar Pesos"
    • Esconder categorias e excluir categorias do cálculo: "Configurar Pesos"
    • Definir a escala de avaliação e/ou as letras correspondentes: "Configurar Letras"
    • Excluir tarefas do cálculo da avaliação de um estudante: "Exceções"
