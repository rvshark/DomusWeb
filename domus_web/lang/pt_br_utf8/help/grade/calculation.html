Cálculos

Cálculos para o relatório de notas seguem o padrão de fórmulas/funções de programas de planilhas. Eles começam com um sinal de igual(=), e usam operadores e funções matemáticas comuns para produzir uma única solução numérica. Esta saída é então usada como o valor computado para o item de avaliação que você está editando.

Segue uma lista das funções suportadas pelo cálculo. A vírgula(,) é usada para separar argumentos dentro de parênteses.

  • média([[item1]], [[item2]]...):Retorna a média de uma amostra
  • max([[item1]], [[item2]]...): Retorna o valor máximo em uma lista de argumentos
  • min([[item1]], [[item2]]...): Retorna o valor mínimo em uma lista de argumentos
  • mod(dividendo, divisor): Calcula o resto de uma divisão
  • pi(): Retorna o valor do número pi Pi
  • potência(base, potência): Eleva um número a potência de outro
  • Arredondamento(número, casas decimais): Arredonda um número a uma precisão predefinida
  • soma([[item1]], [[item2]]...): Retorna a soma de todos os argumentos

Você pode pode incluir os valores dos outros itens de avaliação usando seus números de id como referências em suas fórmulas. O número de id é cercado por dois pares de colchetes para evitar confusão na análise do cálculo quando o número de id contém caracteres incomuns. Por exemplo, se você tem um item de avaliação com Questionário.3 como um número de id, você vai se referir a este item como [[Questionário .3]] em seu cálculo.

Abaixo da área de cálculo tem um esquema em cascata do seu curso com suas categorias e itens de avaliação. Junto do total de cada item ou categoria é mostrado o número de id que você pode usar em seu cálculo(já entre colchetes). Como o número de id é opcional, alguns itens ainda podem não ter um. Esses itens sem um número de id têm no lugar um campo de formulário onde você pode digitar o número de id diretamente, e assim usá-lo no seu cálculo sem ter que sair da página. Esse formulário não permite que você edite números de id existentes, você vai ter que editar o item de avaliação diretamente se quiser mudar.

Assim que você tiver designado os números de id necessários, pode clicar em "adicionar números de id", e a página vai carregar novamente e mostrar o mesmo esquema em cascata com os números id que você acabou de designar. Você agora está livre para usar estas referências em seus cálculos.

Aqui tem uma lista de exemplos práticos de cálculos que você pode usar:

  • =média([[Questionário.1]], [[Questionário.4]], [[Tarefa.1]])
  • =média(max([[Questionário.1]], [[Questionário.4]], [[Tarefa.1]]), min([[Questionário.1]], [[Questionário.4]], [[Tarefa.1]]))
