Chave de usuário

Publicação de notas é uma maneira de exportar notas através de uma URL sem estar logado ao Moodle. Uma chave de usuário é uma série de caracteres gerada aleatoriamente, que é anexada a URL por razões de segurança.

Para publicar notas, selecione "Criar uma nova chave de usuário". Quando o formulário for enviado, uma nova chave de usuário é criada. Alternativamente, uma chave que foi criada previamente pode ser selecionada.
