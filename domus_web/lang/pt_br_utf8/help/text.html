Formataçãoo de Textos Simples

Além de escrever textos como em outros editores, você pode incluir "smilies", "endereços URL" e algumas etiquetas HTML.

Smilies (emoticons)

    Para incluir um destes símbolos no seu texto, escreva o código correspondente. Estes códigos parecem carinhas desenhadas se você olhar para eles com a sua cabeça virada para a esquerda.

    smile :-)
    big grin :-D
    wink ;-)
    mixed :-/
    thoughtful V-.
    tongue out :-P
    cool B-)
    approve ^-)
    wide eyes 8-)
    surprise 8-o
    sad :-(
    shy 8-.
    blush :-I
    kisses :-X
    clown :o)
    black eye P-|

    evil }-]

URLs

    Qualquer palavra que começar com www. ou http:// será transformada, automaticamente, em link.

    Por exemplo: www.yahoo.com ou http://curtin.edu

Código HTML

    Você pode usar um número limitado de etiquetas HTML para dar ênfase ao seu texto.

    HTML tagsProduces
    <b> bold </b> bold text
    <i> italic </i> italic text
    <u> underline </u> underlined text
    <font color="green"> example </font> example
    <ul> <li>one</li> <li>two</li> </ul>
    • one
    • two
