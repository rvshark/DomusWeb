Filtro do Perfil

Este filtro permite filtrar usuários baseando-se em campos do perfil. O filtro pode ser aplicado em um único campo do perfil ou em todos os campos do perfil. O filtro tem as seguintes opções:

  • contém - esta opção seleciona somente usuários que tem o campo especificado contendo o texto fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • não contém - esta opção seleciona somente usuários que não tem o campo especificado com o texto fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • é igual a - esta opção seleciona somente usuários que tem o campo especificado com o texto igual ao fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • começa com - esta opção seleciona somente usuários que tem o campo especificado começando com o texto fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • termina com - esta opção seleciona somente usuários que tem o campo especificado terminando com o texto fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • está vazio - esta opção seleciona somente usuários que tem o campo especificado com o texto fornecido vazio mas que foi definido (o texto fornecido é ignorado)
  • não está definido - esta opção seleciona somente usuários que não tem o campo especificado definido (o texto fornecido é ignorado)
  • está definido - esta opção seleciona somente usuários que tem o campo especificado definido (o texto fornecido é ignorado)
