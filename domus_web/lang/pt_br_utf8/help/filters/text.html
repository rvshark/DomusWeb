Filtro Texto

Este filtro permite filtrar informações baseando-se em um texto em formato livre. O filtro tem as seguintes opções:

  • contém - esta opção seleciona somente informações que contem o texto fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • não contém - esta opção seleciona somente informações que não contem o texto fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • é igual a - esta opção seleciona somente informações com o texto igual ao fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • começa com - esta opção seleciona somente informações começando com o texto fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • termina com - esta opção seleciona somente informações terminando com o texto fornecido (se o campo de texto estiver vazio, então o filtro não é habilitado)
  • está vazio - esta opção seleciona somente informações que sejam vazias (o texto fornecido é ignorado)
