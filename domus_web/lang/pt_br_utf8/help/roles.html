Funções

Uma Função é uma coleção de permissões que você pode designar a n~ivel de sitema ou em contextos específicos.

Por exemplo, você pode definir uma função chamada "Professor" que é configurada para permitir aos professores que façam certas coisas ( e não façam outras). Uma vez que esta função exista, voce pode designa-la para alguem em um curso para torna-lo um "Professor" dentro deste curso. Voce pode também designa-lo para a função na categoria do curso para torna-lo "Professor" em todos os cursos dentro desta categoria, ou designar a função para um usuário sómente dentro de um único forum, dando a este usuário aquelas capacidades só neste forum.

Uma função tem que ter um nome. Se você precisar nomear uma função em vários idiomas use a sintaxe para multiplos idiomas da seguinte forma: 

  <span lang="en">Teacher</span> 
  <span lang="es_es">Professor</span>
Se você fizer isto verifique se a configuração "filtro de cadeia de caracteres" está ativada em sua instalação.

O nome abreviado é necessário para outros plugins no Moodle que podem precisar fazer referência a sua Função (p.ex. carregando usuários de um arquivo ou fazendo inscrições via plugin).

A descrição explica a função em suas palavras, de forma que todos possam entender a função.
