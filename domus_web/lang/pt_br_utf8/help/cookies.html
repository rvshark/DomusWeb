Cookies

Este site usa dois cookies.

O cookie mais importante é o da sessão, normalmente chamado MoodleSession. Você deve autorizar o seu navegador a aceitar este cookie para que o seu acesso seja contínuo e não se interrompa durante a navegação. Quando você sai do site ou fecha o seu navegador este cookie é automaticamente eliminado.

O outro cookie serve, por pura comodidade, para memorizar o seu nome de usuário no navegador. Normalmente este cookie se chama MOODLEID. Isto significa que quando você voltar a este site o nome de usuário vai estar preenchido no formulário de acesso. Se você não aceitar este cookie a única consequência vai ser a necessidade de preencher manualmente o formulário sempre que você fizer o login.
