Tempo determinado

Essa opção coloca um limite de tempo para cada lição. É mostrado aos estudantes um contador em JavaScript e o tempo é gravado no banco de dados. Devido à natureza inconsistente do JavaScript, o cronômetro não faz com que o aluno saia da lição quando acabar o tempo, mas as respostas enviadas após esse momento não serão registradas. O tempo no banco de dados é checada toda vez que o estudante envia uma questão.
