Mostrar Menu à Esquerda

Isto mostrará uma lista das páginas (Painel de Navegação) na lição.

Especificando uma nota maior que 0, o usuário que está seguindo a Lição deve ter uma nota igual a ou maior que a nota estabelecida para que possa ver o Menu Esquerdo. Isto permite que os autores de Lições forcem os usuários a percorrer toda a lição na sua primeira tentativa. Então, se um usuário retomar a Lição após obter a nota exigida, ele poderá ver o Menu à Esquerda para ajudar na revisão.
