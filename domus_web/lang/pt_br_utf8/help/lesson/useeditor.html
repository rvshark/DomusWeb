Uso do Editor HTML

Quando editando páginas esta opção habilita o uso do editor HTML (de agora em diante chamado de editor) ao invés de caixas de área de texto simples. O editor permite a formatação simples do texto e a fácil adição de características "avançadas" tais como links.

O editor é ativado ao se clicar na caixa próxima a Usar Editor. O editor é associado com os itens de forma individual. Por exemplo, ativando o editor para, digamos, o item Retorno 1 apenas ativa o editor para este item, ele não é ativado para todas as caixas de Retorno.

Se o editor é solicitado para alguns itens de uma página, primeiro clique nas caixas Usar Editor (Então elas estarão marcadas.) A página é então mostrada novamente para ativar o uso do editor. Isto é feito ao clicar no botão Reapresentar Página ao pé da página.

Note que o editor deve ser usado com cuidado. Na maioria dos casos ele não deve causar problemas. Mas, nãot é recomendado para Questões do tipo Respostas de Resposta Curtas e Numéricas. O editor pode adicionar etiquetas HTML "invisíveis" extras que fazem parar o trabalho de comparação esperado. Além disto, a Descrição na Tabela de Desvios são usadas em Botões e outra vez se contiverem qualquer etiqueta HTML isto poderá causar problemas. Não há problemas com respostas que são usadas sómente para mostrar, como aquelas de questões do tipo múltipla escolha. Outra vez o uso do editor para itens de Retorno não devem causar problemas.

Se existe suspeita de problema com qualquer dos itens de texto é seguro desligar o editor (removendo a marcação da caixa Usar Editor ). Quando a página é reapresentada, o texto "puro" poder ser visto na caixa de área de texto. O texto pode então ser editado. De fato, alternando a ativação e desativação do editor várias vezes para um item não deverá mudar o texto.

A "escolha" da opção Usar editor é guardada com cada item. Assim se uma página for reeditada os itens que estavam usando o editor irão de novo aparecer dentro do editor.

Finalmente, o editor é um grande "gastador" de recursos dentro do navegador. Se alguns itens não necessitam do uso do editor é mais rápido e mais economico de recursos usar simplesmente a caixa de área de texto padrão.
