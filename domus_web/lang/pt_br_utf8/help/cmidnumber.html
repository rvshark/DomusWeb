Número ID

O Número ID identifica a atividade para fins de cálculo de avaliação. Se a atividade não estiver inclusa em nenhum cálculo de avaliação então o campo do Número ID pode ser deixado em branco.

O Número ID também pode ser definido na página de edição do cálculo das notas no Relatório de Avaliação, embora ele só possa ser editado na página de atualização da atividade.
