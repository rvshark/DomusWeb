Esta configuração determina se os estudantes podem se auto-inscrever no curso usando o plugin interativo de inscrição padrão (ex. Inscrição Interna). Em caso contrário eles receberão a mensagem "Este curso não aceita inscrições no momento."

Esta configuração não tem efeito sobre os plugins não-interativos de inscrição (ex. Inscrição a partir de banco de dados externo). Além disto, os usuários podem ser designados para a função de estudante através do link Designar Funções no bloco de administração do curso.
