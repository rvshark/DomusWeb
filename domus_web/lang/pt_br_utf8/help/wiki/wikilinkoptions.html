Opções de Auto-linking

Se você desejar que os links automáticos a partir de palavras CamelCase não sejam criados, selecione este box para desabilitar esta função.

ATENÇÃO -- O CamelCase é uma característica típica dos wikis. Disabilitando isto os wikis importados podem ter problemas de funcionamento. Use esta opção apenas se você estiver certo que o CamelCase não é desejável.
