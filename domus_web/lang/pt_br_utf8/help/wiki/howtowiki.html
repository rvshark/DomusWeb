Como criar um Wiki

A atividade wiki do Moodle é baseado no ErfurtWiki,

Parágrafos

  • separe parágrafos dentro de um texto com linhas em branco
  • use três sinais de percentual %%% para forçar uma quebra de linha
  • caso você inicie o texto com espaços ou tabulações antes dos textos, eles serão reconhecidos como sinais de identação

!! Cabeçalho

  • use uma marca de exclamação ! no início de uma linha para criar um pequeno cabeçalho
  • !! para um cabeçalho médio
  • !!! para um grande

Estilo de Texto

  • se você quizer enfatizar um texto, coloque o texto entre duas aspas simples '' (o texto aparecerá em itálico)
  • o texto fica em negrito com dois sublinhados __ (ou se estiver entre dois asteriscos **)
  • para fazer um texto maior coloque o texto entre cerquilhas ##
  • você pode fazer um texto menos utilizando "µµ" da mesma forma
  • uma fonte tipo máquina de escrever caso você coloque o texto entre sinais de igual ==

Listas

  • inicie uma linha com * para iniciar uma lista
  • use # para criar uma lista numerada
    1. você pode criar sublistas
    2. subseqüências de listas devem começar com a mesma combinação de * e #

HyperLinks

  • entre com um WikiWord em seu texto para criar um HyperLink
  • ou, coloque a palavra entre chaves para criar um HyperLink em uma palavra que não seja uma WikiWord válida
  • qualquer endereço válido de internet (começando com http://) como http://www.example.com/ se transforma em um link automaticamente.
  • coloque um endereço www ou um link Wiki entre chaves [Moodle] e atribua a ele um título puro utilizando aspas ou o caractere |
    • [título | http://exemplo.com]
    • ["título de um" WikiWord ] ou ["título de" um WikiLink]
  • se você não quer uma WikiWord ou um !http://www-endereco (ou [qualquer coisa] dentro de chaves) para criar um HyperLink então proceda o texto com um sinal de exclamação ou o acento til (~).
    • !SemHyperLink, ~SemHyperLink
    • ![sem hyperlink], !!http://semlink.org/

Tabelas com |

| apenas coloque | as palavras entre | traços |
| para construir | uma estrutura de tabela |
| browsers normalmente | deixam fora as | células faltantes |

O resultado desse código será:

apenas coloque as palavras entre traços
para construir uma estrutura de tabela
browsers normalmente deixam fora as células faltantes

Sempre coloque uma linha em branco antes e depois da tabela, desta forma ela estará fora de outros textos no seu próprio parágrafo.

Fotos

  • para incluir uma imagem em uma página coloque o endereço absoluto da imagem entre chaves, como [http://www.exemplo.com.br/imagens/imagem.png]
  • alternativamente você pode utilizar a opção de inserir de imagens

Leituras adicionais

Existem mais possibilidades para marcações Wiki. Favor consulte Página Wiki de Erfurt para maiores informações.

CamelCase:
CamelCase descreve como WikiWords parecem. Palavras múltiplas, unidades sem espaços, separadas apenas por letras maiúculas. As letras maiúsculas e minúsculas se assemelham a corcovas de camelos.

Este esquema de links é chamado frequentemente de BumpyText.

Enquanto os puritanos da língua odeiam o esquema de nomenclatura do Wiki, ele é muito utilizado no mundo da computação e bem conhecida pelos programadores.
