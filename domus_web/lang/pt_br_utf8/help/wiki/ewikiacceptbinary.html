Permitir arquivos binários

Se você permitir o uso de arquivos binários (como imagens), você tem duas possibilidades:

  1. Carregar no servidor e usar as imagens em páginas do wiki. Quando estiver editando uma página, um campo para o upload de imagens será disponível. Depois de carregar a imagem no servidor, um código será visualizado para que você possa incluir a imagem na página, utilizando parênteses quadradas. Por exemplo: [internal://myimage.gif].
  2. Você também pode anexar arquivos em uma página wiki. Esses serão visualizados utilizando a função anexos. A dimensão permitida depende dos limites configurados em Moodle.
