Configurar indicadores das páginas (flags)

Os Flags são associados a uma página wiki para mudar o tipo ou comportamento.

Descrição dos Flags:

TXTA página contém texto.
BINA página contém dados binários.
OFFA página está disabilitada.
HTMPermitir HTML nesta página (a configuração geral do wiki tem preferência).
ROApenas leitura.
WRPermitido editar.
