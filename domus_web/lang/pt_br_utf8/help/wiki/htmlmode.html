Modalidade HTML

Existem as seguintes opções:

Nenhum HTML
Ignora o código HTML. A formatação é baseada em WikiWords. Os filtros são habilitados.
HTML Seguro
Algumas Tags HTML são permitidas. Os filtros que dependem de marcadores não são habilitados !
Apenas HTML
Nenhuma WikiWord, apenas HTML. Esta opção ativa o editor HTML.
