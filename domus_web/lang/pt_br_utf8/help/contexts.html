Contextos

Contextos são áreas lógicas do Moodle que definem o raio de ação e as permissões das funções designadas aos usuários.

Eles são organizados de forma hierárquica, com permissões herdadas dos contextos "mais abrangentes" para os "menos abrangentes".

  1. Sistema (toda a instalação)
  2. Página Inicial (o "curso do site")
  3. Categorias de Curso
  4. Sub-categorias de Cursos
  5. Cursos
  6. Blocos e Atividades

Veja também: Funções, Permissões, Designar Funções, Sobreposições.
