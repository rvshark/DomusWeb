Como Fazer Buscas

A busca em textos completos suporta as várias opções listadas abaixo. Você pode combiná-las para especificar sua busca com maior precisão.

buscar estas palavras Para a busca básica por uma ou mais palavras em qualquer parte dos textos, simplesmente digite-as separadas por espaços. Todas as palavras com mais de dois caracteres são usadas.
+buscar +estas palavras O exemplo anterior encontraria também a palavra "festas" pois contém "estas". Para forçar a busca de uma palavra exata, use o sinal de mais.
+busca -motor Use o sinal de menos se houverem palavras exatas específicas que você não quer que sejam incluídas na busca.
"motor de busca" Para buscar uma frase específica, use a frase cercada por aspas duplas.
user:Kim Para buscar textos de um usuário específico, use o prefixo "user:" seguido pelo nome dele.
userid:6 Se você conhece o id de um usuário em específico, você pode buscar por ele desta forma.
subject:avaliação Para buscar uma palavra apenas se contida no assunto ou no título de um texto, use o prefixo "subject:" antes da palavra

Para busca avançada, pressione o botão de busca sem digitar nada no campo de palavras  você verá um formulário completo que torna mais fácil fazer buscas avançadas.
