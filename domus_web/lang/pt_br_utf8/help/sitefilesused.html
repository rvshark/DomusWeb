Arquivos do Site usados nesse curso

Essa opção permite que você escolha se quer fazer backup dos arquivos do site em um curso.

Atualmente os únicos arquivos que são vistos como usados em um curso são :

  • arquivos ligados a questões num banco de questões.
  • Todos os arquivos na área de arquivos do site que são ligados à questões armazenadas também são guardados, caso você escolha 'sim' nessa opção.
