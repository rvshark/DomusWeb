Tornar-se Proprietário de um Exercício

Este link permite ao professor tornar-se proprietário do conjunto de envios do exercício atual. Você precisará fazer isto se importou um exercício de um backup de outro curso. Este link permite a requisitar todas os envios do exercício. No curso atual estes envios importados poderiam pertencer a qualquer um, até mesmo a um estudante. Este procedimento evita este problema.

Além da propriedade, o s enviosnão são alterados por este procedimento.
