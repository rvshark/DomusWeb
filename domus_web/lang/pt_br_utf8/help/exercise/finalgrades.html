As Notas Finais

A tabela nesta tela lista as notas e seus desdobramentos. A primeira nota é para a avaliação (inicial). Esta é adicionada as notas dadas a cada envio. A nota final dada a esta tarefa será ou a média destas notas ou a nota do melhor envio.

A tabela Notas mais altas opcionalmente lista os melhores trabalhos. O número de entradas na tabela é configurado em um dos parametros do Exercício. Se este número é igual a zero então a tabela Notas mais altas não será mostrada. A tabela está ordenada pelas notas com o melhor trabalho primeiro. Sómente o melhor trabalho de cada estudante é mostrado na tabela.
