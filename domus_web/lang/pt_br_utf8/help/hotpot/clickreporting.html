Ativar relatório de cliques

Sim
Um registro separado é armazenado no banco de dados cada vez que botões de "dica", "pista" ou "checagem" são clicados. Isso permite que o professor tenha um relatório detalhado dos acontecimentos do quiz a cada clique. Entretanto, isso pode significar um aumento rápido do tamanho do banco de dados, porque vários registros de "clique" podem ser gravados a cada tentativa.

Não
Apenas um registro por tentativa no quiz é gravado no banco de dados do Moodle.

Por padrão a opção é desativada.
