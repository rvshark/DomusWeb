Formatos de relatório

Você pode especificar diferentes formatos de relatórios do Hot Potatoes.

Formato do Relatório

HTML
o relatório será gerado em HTML, para a exibição em um navegador web

Excel
o relatório será gerado como um arquivo ".xls", que pode ser aberto num programa de planilhas, como o Microsoft Excel

Texto
o relatório será gerado como um arquivo de texto, que pode ser aberto por um editor de texto

Codificação

Uma lista de codificações é oferecida, o que permite forçar o relatório para uma codificação específica, caso seu programa de planilhas exija. Por exemplo, o Microsoft Excel requer a codificação "SJIS" para o japonês. A lista de codificações possíveis pode ser ajustava pelo administrador do site na página de "Configurações" do módulo HotPot

Pular linha

Sim
dados muito longos terão quebras de linha para que as tabelas fiquem arrumadas. Isso podem deixar algumas linhas muito altas

Não
os dados não terão quebra de linha

Mostrar legenda

Sim
no relatório principal, questões e respostas serão representadas por letras. Uma legenda será gerada para traduzir as respectivas representações.

Não
questões e respostas aparecerão por completo no relatório, e nenhuma legenda será gerada
