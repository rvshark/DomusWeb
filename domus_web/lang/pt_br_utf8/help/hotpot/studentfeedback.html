Feedback do estudante

Se esssa opção estiver ativada, um link para um popup de janela de feedback será exibido sempre que o estudante clicar no botão "Check". A janela de feedback permite que o estudante envie um feedback para o professor a respeito do quiz.

Essa opção pode ter as seguintes configurações:

Nenhum
Nenhuma janela de feedback será exibida

Página Web
Uma página Web será exibida na janela de feedback. Você vai precisar inserir uma URL para a página web, por exemplo: http://myserver.com/feedbackform.html

Formulário de feedback
Um formulário adequado para ser enviado para um script formulário-para-email padrão, tais como formmail.pl, será exibido na janela de feedback. Você também deve inserir a URL para o script do formulário, por exemplo: http://myserver.com/cgi-bin/formmail.pl

Fórum do Moodle
O índice do fórum para este curso do Moodle será exibido

Envio de mensagens do Moodle
A janela de envio de mensagens instantâneas do Moodle será exibida. Se o curso tiver vários professores, o estudante será levado a selecionar um professor antes que a janela de envio de mensagens apareça.
