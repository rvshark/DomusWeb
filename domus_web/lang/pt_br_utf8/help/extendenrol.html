Estender período de inscrição

Esta página permite modificar o período de inscrição de um participante ao curso. O novo período de inscrição é calculado a partir de uma data de referência de início e um período de extensão a partir desta data. Dependendo desta combinação você pode restringir ou extender o período de inscrição dos participantes.

As datas de referência de início disponíveis são:

  • Data de início do curso
  • Início de Inscrição - a data de início de inscrição do participante (esta opção está disponível somente para usuários com data de início de inscrição conhecida)
  • Término de Inscrição - a data de término de inscrição do participante (esta opção está disponível apenas para usuários com período limitado de inscrição)
  • Hoje (esta opção está disponível somente se a data de início do curso já passou)
  • Data de início do período de inscrição ao curso - página de configuração da data de início do período de inscrição ao curso (esta opção está disponível somente para cursos com data de início de inscrição definida)
  • Data de fim do período de inscrição ao curso - página de configuração da data de fim do período de inscrição ao curso (esta opção está disponível somente para cursos com data de fim de inscrição definida)

Se o período de extensão é configurado para 'Não mudado' ou 'Sem título', então a data de referência de início é ignorada.
