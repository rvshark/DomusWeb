Parametros IMS

  • Menu de navegação lateral - Lista de links à esquerda do conteúdo
  • Lista de conteúdos (TOC) - Lista de links na página inicial, e um botão TOC (quando os botões de navegação são mostrados)
  • Botões de navegação - Botões Anterior e Próximo,com opção para botão TOC e Para Cima
  • Saltar páginas de sub-menu - Omite navegação de sub-menus
  • Botão Para Cima - Botão para retornar à seção de navegação
