Remover estudantes/professores

Dados Enterprise podem acrescentar ou também remover inscrições dos estudantes e professores nos cursos. Se essa opção estiver ativada, o Moodle seguirá as indicações de remoção de usuários.

Há três modos de remover os estudantes usando dados IMS:

  • Com um elemento <member>, que indica o estudante e o curso, e com o atributo "recstatus" do elemento <role> definido como 3 (que significa "apagar'). ISSO AINDA NÃO FOI IMPLEMENTADO NO PLUGIN DO MOODLE.
  • Com um elemento <member>, que indica o estudante e o curso, e com o elemento <status> definido como 0 (que significa "inativo").

O terceiro método é um pouco diferente. Ele não requer que essa configuração de remover estudantes esteja ativa, e pode ser programado com antecedência em relação à data de remoção:

  • Com um elemento <member> que especifica um <timeframe>, para a definição das datas de ínicio e/ou término da permanência do estudante. Essas datas são carregadas na tabela de dados de inscrição do Moodle (se presente), e então, após uma data final, o estudante não poderá mais entrar no respectivo curso.
