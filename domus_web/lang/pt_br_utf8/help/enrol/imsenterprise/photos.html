Fotos

Quando é criada uma nova conta de usuário a partir de dados Enterprise, é possível que uma foto tenha sido especificada pelo usuário - normalmente fornecendo uma URL para o arquivo da imagem.

Se o dado <photo> é fornecido, então o Moodle pode tentar importar o arquivo da imagem como foto do usuário no Moodle.

Atenção: Isto pode adicionar uma sobrecarga ao processo se houverem muitas contas de usuários, por causa da necessidade de transferir, modificar e armazenar as imagens. Não é recomendado ativar esta opção quando o número de contas a serem criadas é muito grande.
