Carregar grupos

Esse recursos permite a criação de grupos a partir de um arquivo.

  • Cada linha contém um registro
  • Cada registro é uma sequência de dados separados por vírgulas
  • O primeiro registro do arquivo é especial, e contém uma lista de campos. Isso define o formato do resto do arquivo.

    Campos Obrigatórios: esses campos precisam ser incluídos na primeira linha, e definidos para cada usuário

    groupname

    Nomes dos campos padrão: esses são opcionais - se não forem incluídos então os valores usados vêm do idioma atual e do curso

    idnumber, coursename, lang

    Nomes dos campos opcionais: todos esses são completamente opcionais.

    description, enrolmentkey, picture, hidepicture

  • As vírgulas usadas nos dados precisam ser codificadas como &#44 - o script irá decodificar automaticamente.
  • Para campos booleanos(lógicos), use 0 para falso e 1 para verdadeiro.
  • Idnumber ou coursename podem ser usados para identificar o curso. Idnumber sobrepõe coursename. Se nenhum dos dois for especificado, os grupos serão adicionados ao curso atual.
  • Coursename é o nome reduzido do curso.
  • Nota: Se um grupo já estiver registrado no banco de dados do Moodle em um curso, esse script reconhecerá o nome do grupo. Professores só podem criar grupos nos cursos que eles podem editar.

Aqui um exemplo de arquivo de importação válido:

groupname,idnumber,lang,description,picture
grupo1, Joao01, pt_br_utf8, este grupo é especial!, 0
grupo12, Mat243, , ,
