Exportação de perguntas de uma Categoria

Esta função permite exportar uma categoria completa de perguntas para um arquivo de texto.

Por favor, preste atenção que em muitos formatos de arquivos se perdem algumas informações quando as perguntas são exportadas. Isto acontece quando os formatos não possuem as características que existem nas perguntas Moodle. Ao importar e exportar perguntas não julgue que elas são idênticas. Pode mesmo acontecer de não ser possível exportar alguns tipos de perguntas. Aconselha-se que sejam verificados os dados exportados antes da utilizsação num contexto de produção.

Os formatos aceitos atualmente são:

Formato GIFT

    GIFT é o formato de importação/exportação mais completo que existe à disposição para exportar perguntas de questionários Moodle para um arquivo. Foi criado para facilmente os professores escreverem perguntas num arquivo. Aceita Múltipla-Escolha, Falso-Verdadeiro, Resposta Breve, Associação e Questões Numéricas, bem como a inserção de _____ para a palavra que falta. Repare que as perguntas "Cloze" não são normalmente aceitas. Podem-se misturar vários tipos de perguntas num único arquivo e o formato também aceita comentários, nomes de perguntas, "feedback" e avaliações a nível de porcentagem. Seguem-se alguns exemplos:

    Quem está enterrado no túmulo de Grant?{~Grant ~Jefferson =ninguém}

Grant está {~enterrado =sepultado ~vivo} no túmulo de Grant.

Grant está enterrado no túmulo de Grant.{FALSO}

Quem está enterrado no túmulo de Grant?{= nenhuma pessoa = ninguém}

Quando nasceu Ulysses S. Grant?{# 1822}

    Informação adicional sobre o formato GIFT

Formato Moodle XML

    Este formato específico do Moodle exporta perguntas num formato XML simples. As perguntas assim exportadas podem ser importadas para outra categoria de perguntas, ou usadas em outros processos tais como transformações XSLT.

IMS QTI 2.0

    Exporta usando o formato padrão IMS QTI (versão 2.0). Repare que serão gerados um grupo de arquivos dentro de um único arquivo 'zip'.

    Mais informações encontram-se no site do IMS QTI (site externo numa nova janela)

XHTML

    Exporta a categoria como uma única página com XHTML 'estrito'. Cada pergunta é colocada dentro da sua própria etiqueta <div>. Se quiser usar essa página tal qual, precisará pelo menos editar a etiqueta <form> no início da seção <body>, para que seja realizada alguma ação apropriada (por exemplo, enviar uma mensagem, usando 'mailto').

Mais formatos deverão aparecer, incluindo WebCT e qualquer outro que os utilizadores do Moodle possam contribuir!
