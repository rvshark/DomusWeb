Importando arquivos no formato"Moodle XML"

Transferindo Categoria

Pode-se inserir um questão falsa especial no arquivo XML para especificar a hierarquia de categorias na qual as questões seguintes serão inseridas. Se a categoria ou categorias não existirem elas serão criadas. Note que a opção do arquivo: deve estar marcada na página de opções ou este código será ignorado completamente. Pode-se transmitir categorias no arquivo xml quantas vezes se queira.

Segue um exemplo do formato:

<question type="category">
    <category>
        tom/dick/harry
    </category>
</question>
