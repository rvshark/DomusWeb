Download da Sinopse

Você pode querer baixar os dados mostrados na tabela da tela para análise posterior. Pode-se escolher entre dois formatos de arquivo para os dados baixados.
Em cada caso os dados serão apresentados como uma tabela com os títulos apropriados, como na tela. Se a tabela estiver paginada, todos os dados serão baixados em um único arquivo.

Formato de planilha Excel:

Você obterá uma planilha .xls.

Formato texto

Neste caso, os dados serão gravados como um arquivo texto. Uma linha para cada coluna da tabela com os dados separados por tabuladores.
