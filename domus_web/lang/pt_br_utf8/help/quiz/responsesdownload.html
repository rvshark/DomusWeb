Descarregar respostas

Você pode desejar descarregar os dados da tabela mostrados na tela. Você pode escolher entre dois formatos de arquivos para os dados a serem descarregados.
Em cada caso os dados serão apresentados como uma tabela com títulos apropriados, como na tela mas sem as figuras do usuário.
Se a tabela é paginada ou ordenada na tela, todos os dados para todas as páginas serão sempre descarregados em um arquivo simples. Usuários com nenhuma tentativa para este questionário serão incluídos no arquivo se você ativar a opção correspondente.

Formato em planilha Excel:

Você obterá um documento com a extensão .xls.

Formato texto

Neste caso, os dados serão armazenados como um arquivo texto regular. Uma linha para cada coluna na tabela com os dados separados por tabuladores.
