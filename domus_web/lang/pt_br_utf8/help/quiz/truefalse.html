Questões Verdadeiro/Falso

Para responder a questão (que pode incluir uma imagem), o participante escolhe entre as opções Verdadeiro ou Falso.

Se o questionário compreender notas de feedback, estas serão visualizadas após o envio das respostas. Por exemplo, se a resposta correta é "Falso" mas a reposta dada é "Verdadeiro", o feedback definido para a opçãoo "Verdadeiro" será visualizada.
