Conjunto de dados para Perguntas

Um conjunto de dados é uma coleção de dados utilizados para criar uma pergunta (tal como uma questão de cálculo), onde é inserido no lugar de uma variável dentro da pergunta.

Pode-se criar um conjunto de dados "privado" para uma pergunta específica, ou um conjunto de dados "reutilizável" que pode ser utilizado dentro de todas perguntas dentro de uma categoria.

Quando se criar uma pergunta que utiliza conjuntos de dados, serão apresentadas duas telas.

A primeira tela permite acessar um conjunto de dados particular para ser submetido para cada variável.

A segunda tela permite acrescentar e remover conjuntos de números no conjunto de dados. Os números utilizados para as perguntas individuais apresentadas aos estudantes serão selecionados deste conjunto.
