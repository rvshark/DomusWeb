Respostas embutidas (Cloze)

Este tipo muito flexível de pergunta é similar ao formato popular conhecido como formato Cloze.

As perguntas consistem em uma passagem de texto (no formato Moodle) que tem várias respostas embutidas dentro dela, incluindo escolha múltipla, respostas breves e respostas numéricas.

Não existe uma interface gráfica para criar estas perguntas - deve-se especificar o formato da pergunta utilizando-se uma caixa de texto ou importando de arquivos externos.

Aqui está um exemplo de um texto de entrada utilizado para especificar tal pergunta:

Este exemplo aparecerá aos estudantes como se segue:

3

8 Marcas

Esta pergunta consiste em alguns textos com uma resposta embutida exatamente aqui e exatamente depois disto deverá lidar com esta resposta breve e finalmente tem-se um número com ponto flutuante .

Note que endereços como http://www.moodle.org/ e smileys trabalham normalmente:
a) Isto é bom?
b) Qual a nota que você daria a isto?

Boa sorte!
