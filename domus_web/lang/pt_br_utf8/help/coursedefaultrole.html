Função Padrão no curso

Esta função é atribuída aos usuários quando nenhuma outra é definida, por exemplo, quando os usuários se increvem no curso usando o código de inscrição.

É também usada como padrão por rotinas de cadastramento que inscrevem participantes no curso a partir de dados de outros sistemas.

Na maioria das vezes é usado o Padrão do Site asdefinido pelo Administrador do Sistema para todos os cursos.
