Tipo de Trabalho

Há vários tipos de trabalho

Trabalho de casa - é útil quando o trabalho é executado fora do Moodle. Os alunos podem ver uma descrição do trabalho, mas não podem enviar ficheiros nem nada. No entanto, os professores podem avaliar todos os alunos e estes terem acesso às suas avaliações.

Envio de um único ficheiro - este tipo de trabalho permite que cada aluno envie um ficheiro de qualquer tipo. Pode ser um documento WORD, uma imagem, um sítio web zipado ou algo que lhes peça para submeter. Os professores podem avaliar estes trabalhos online.
