Incluindo Temas

Você poderá utilizar este link para incluir um Tema no diálogo. É uma boa idéia sempre dispor de um Tema para o diálogo, isto mantém o diálogo no foco do Tema e direciona as respostas para o tópico. Se você quiser falar sobre outro Tema é melhor fechar o diálogo atual e iniciar um novo diálogo no novo tópico.
