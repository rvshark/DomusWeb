Atividades de Banco de Dados

O módulo de Banco de Dados permite ao professor e/ou aos alunos criar, mostrar e buscar um banco de dados sobre qualquer tópico. O formato e a estrutura dos itens sçao quase ilimitados, incluindo imagens, arquivos, URLs, números e texto. Você pode estar familiarizado com uma tecnologia parecida na construção de bancos de dados com Microsoft Access ou Filemaker.
