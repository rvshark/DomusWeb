Preço do Curso

Se este site foi configurado para usar um método de inscrição que solicite pagamento (como o módulo Paypal), então você pode colocar o custo aqui, sem símbolo algum, por exemplo: 19,95. A moeda corrente é ajustada automaticamente pelo plugin.

Se o campo custo estiver vazio, então a opção pagamento não é apresentada e a interface será direcionada para outros métodos de inscrição (como chaves de acesso, ou inscrições manuais).

Se o campo custo NÃO estiver vazio, será apresentada a opção de fazer o pagamento para entrar aos estudantes que tentarem registrar-se.

Se você definir TAMBÉM uma chave de acesso nas configurações do curso, os estudantes terão a opção de registrar-se usando uma chave de acesso. É uma opção útil quando há uma mistura de estudantes pagantes e não-pagantes.
