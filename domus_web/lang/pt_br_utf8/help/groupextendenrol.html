Extender a inscrição de todos

Esta página permite extender o período de inscrição de todos os participantes selecionados. O novo período de inscrição é calculado a partir de uma data de referência de início e de um período de extensão a partir daquela data. Dependendo desta combinação é possível restringir o período de inscrição dos participantes.

As datas de referência de início são:

  • Data de início do curso
  • Início da inscrição - a data de início da inscrição do participante
  • Término de inscrição - a data de término de inscrição do participante
  • Hoje (esta opção é disponível se a data de início do curso já passou)
  • Data de início de inscrição no curso - data de início de inscrição preenchida na página de configuração do curso (esta opção é disponível em cursos com data de início de inscrição definida)
  • Data de término de inscrição no curso - data de término de inscrição preenchida na página de configuração do curso (esta opção é disponível em cursos com data de término de inscrição definida)

Se o período de extensão é definido como 'ilimitado', então a data de referência de início é ignorada.
