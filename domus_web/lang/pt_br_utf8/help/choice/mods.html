Escolhas

Este módulo é muito simples: configura uma única pergunta com diversas opções de resposta. Serve para fazer pesquisas rápidas de opinião, para estimular a reflexão sobre um tópico, para escolher entre sugestões dadas para a solução de um problema ou para obter a permissão de utilizar dados pessoais dos alunos em pesquisas do professor. Pode servir também para reservar vagas em grupos ou assuntos diferenciados de discussão.
