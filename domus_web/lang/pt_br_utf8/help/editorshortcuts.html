Teclas de Atalho do Editor HTML

As seguintes opções são acessíveis por teclas de atalho:

Ctrl-O Mudar a fonte
Ctrl-P Mudar o tamanho da fonte
Ctrl-H Mudar o formato do texto (títulos, etc)
Ctrl-1 to Ctrl-6 Mudar o nível do título
Ctrl-= Mudar o idioma do texto para leitores de tela, ou especificar o idioma para o filtro Multilang (eles aparecem na opção 'Multi', na caixa de seleção).
Ctrl-B Negrito
Ctrl-I Itálico
Ctrl-U Sublinhado
Ctrl-S Riscado
Ctrl-, Subscrito
Ctrl-. Sobrescrito
Ctrl-0 Limpar HTML criado pelo Word
Ctrl-Z Desfazer
Ctrl-Y Refazer
Ctrl-L Alinhar à esquerda
Ctrl-E Centralizar
Ctrl-J Justificar
Ctrl-/ O texto segue da esquerda para a direita
Ctrl-| O texto segue da direita para a esquerda
Ctrl-; Diminuir indentação
Ctrl-' Aumentar indentação
Ctrl-G Mudar a cor do texto
Ctrl-K Mudar a cor do fundo
Ctrl-F Localizar e substituir
Ctrl-` Exibição do código HTML
Ctrl-M Trocar para tela cheia
Ctrl-Alt-O Inserir lista ordenada
Ctrl-Alt-U Inserir lista não-ordenada
Ctrl-Alt-R Inserir linha horizontal
Ctrl-Alt-A Criar âncora
Ctrl-Alt-L Criar link
Ctrl-Alt-D Remover link
Ctrl-Alt-N Desativar links automáticos
Ctrl-Alt-I Inserir imagem
Ctrl-Alt-T Inserir tabela
Ctrl-Alt-S Inserir carinha
Ctrl-Alt-C Inserir caractere especial

Limitações atuais:

  • Não é possível usar o atalho Ctrl-` para voltar da exibição de código html para o modo normal.
  • Não há atalhos para edição de tabelas (disponíveis apenas do editor em tela cheia). A edição de tabelas ainda depende do mouse para a seleção das células.
