DFwiki Privileges

In Moodle's DFwiki there are the possibility to manage the accessibility of the users.

So, we can erase privileges from the students inside of DFwiki. We can manage the following privileges:
  • Allow to students edit in DFwiki.
  • Allow to students upload files on server.
  • Allow to students attach files on the wiki pages.
  • Allow to students restore old version from wiki pages.
Edit pages:

If it's checked, the student will be able to read and edit the wiki pages. Else the student only will be able to read. This let the student create new contents.

Upload files:

If it's checked, the student will be able to upload files into DFwiki and they be there avaliables for all the users. Then everybody could include this files in theirs pages.

Attach files:

If it's checked, the student will be able to attach files into wiki pages to give their own format with pictures or gaphics.

Restore old versions:

If it's checked, the student will be able to restore the old versions of a wiki page.
