How ToWiki in syntax DFwiki

In Moodle's DFwiki you can choose the sintax of DFwiki which is an implementation of the team developed by David Castro & Ferran Recio. It allows simple collaborative editing and creation of web pages.

  • learn how to CreatePages
  • select "Edit" from the tabs to change the contents of a page
  • you can format your pages with DFwiki Markup, or with HTML
  • there are diferent kind of blocks to help you to SearchPages or go to the list of NewestPages
  • there are also the options of the MostVisitedPages, MostOftenChangedPages and a large etc.

Create Pages:
You create a page by naming it in the text of an existing page. A wiki page is named enclosing it in square brackets ( [[]] ).

For example:
  • [[My Wiki Page]] (enclosed in square brackets)

Text that has been given a wiki page name will have a '?' after it. Clicking on the '?', will enter edit mode for that page. Enter your text, save and you have a new wiki page.

DFwiki Markup:
Each page inside a Wiki can be edited easily, using dfwiki markup.

Paragraphs

  • separate paragraphs inside the text with empty lines
  • if you prepend text with spaces or tabs it will get indented
  • one space at the beginning of a new line converts all the text with format <pre> until the next carry return

!! Headlines

  • use three equals marks ==== at the beginning and the end of a line to create a large headline, type <h1>
  • == for medium ==, type<h2>
  • = for small headlines =, type <h3>

text style

  • if you want to emphasize text enclose it in two single-quotes ''text'' (it's italic)
  • text gets bold with three single-quotes '''text'''
  • to make an Horizontal line like <hr> we could type ----

Lists

  • start a line with an asterisk * or with hash # to begin a list
  • use * for no-numerated lists
  • use # for numerated lists
    1. you can create sublists
    2. subsequent list points should start with the same mix of * and #

HyperLinks

  • enclose some words in square brackets to create a HyperLink
  • any valid internal page address (starting with [[ and endind by ]] where type is separated by : of the page name and also separated by | of the addicional name)
  • enclose a www address inside square brackets [http://www.moodle.org Moodle]
    • [http://example.com example]
    • [[dfwiki page | addicional name]]
    • [[Name of the page]]
  • if you don't want a WikiWord (or [anything] inside square brackets) to become a HyperLink then just prepend it with <nowiki> and ended by </nowiki>
    • <nowiki>NoHyperLink</nowiki>
    • <nowiki>[no hyperlink]</nowiki>

Tables with |

just enclose text with the exclamation ! when it was headline
and enclose things with the dash character
to build a table structure
browsers usually leave out the missing cells

Please always put an empty line before and after a table, so it stands out from other text in its own paragraph.

Pictures and Files

  • to include an image into a page enclose its absolute www-address in square brackets, like [http://www.example.com/pics/image.png]
  • the type of inserts with external links can be of format: jpg, jpeg, bmp, png, gif y swf
  • alternatively you could use the ImageUploading function and after realize an attach in the page [[attach: name_of_file]].
  • in this way we can put into Dfwiki page all kind of files.
